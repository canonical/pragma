## C3 â€” Naming

| ID | Rule | Gate |
|----|------|------|
| C3.1 | Component names use PascalCase | name matches `/^[A-Z][a-zA-Z0-9]*$/` |
| C3.2 | File names use PascalCase matching the component name | filename = ComponentName + ext |
| C3.3 | Directory names use PascalCase matching the component name | dirname = ComponentName |
| C3.4 | Package-level exports use `index.ts` barrel files | `index.ts` exists |

## C4 â€” Files

| ID | Rule | Gate |
|----|------|------|
| C4.1 | Implementation file: `<Name>.tsx` / `.svelte` / `.ts` | file exists |
| C4.2 | Unit test file: `<Name>.test.tsx` / `.test.ts` | file exists |
| C4.3 | Story file: `<Name>.stories.tsx` / `.stories.ts` / `.stories.svelte` | file exists |
| C4.4 | SSR test file: `<Name>.ssr.test.tsx` / `.ssr.test.ts` | file exists |
| C4.5 | Barrel file: `index.ts` re-exports component and types | file exists |
| C4.6 | Types file: `types.ts` defines component props | file exists |
| C4.7 | Styles file: `styles.css` (optional but conventional) | file exists if styled |

## C5 â€” Exports

| ID | Rule | Gate |
|----|------|------|
| C5.1 | Components are exported from package via barrel files | `src/index.ts` exports component |
| C5.2 | Props types are exported alongside components | types re-exported from index |
| C5.3 | Component index re-exports from implementation and types | `index.ts` has both exports |

### Export Pattern

```typescript
// packages/<pkg>/src/index.ts
export { MyComponent } from "./MyComponent/index.js";
export type { MyComponentProps } from "./MyComponent/types.js";
```

```
[ ] C1.1  Framework is React, Svelte, or Web Components
[ ] C2.1  Component has its own PascalCase directory
[ ] C3.1  Component name is PascalCase
[ ] C4.1  Implementation file exists
[ ] C4.2  Unit test file exists
[ ] C4.3  Story file exists
[ ] C4.5  index.ts barrel file exists
[ ] C4.6  types.ts file exists
[ ] C5.1  Component exported from package barrel
[ ] C6.2  Styles are component-scoped
[ ] C7.4  Vitest is the test runner
[ ] C8.2  Stories use CSF3 format
[ ] C9.1  Props in dedicated types.ts
[ ] C9.3  Strict TypeScript enabled
```
