# Launchpad Components — React

Launchpad is Canonical's software collaboration platform — these components understand its unique data models and interaction patterns. File trees, diff viewers, markdown editors, and UI patterns tailored to code hosting and bug tracking workflows.

## Architecture

The WPE (Web & Platform Engineering) tier extends Apps with product-specific functionality. Components here are highly specialized and may not be suitable for use outside the Launchpad ecosystem.

```
@canonical/react-ds-global (dependency)
         ↓
@canonical/react-ds-app (dependency)
         ↓
@canonical/react-ds-app-launchpad (this package)
         ↓
Launchpad Application
```

## FileTree

Navigate directory structures with expand/collapse and selection:

```tsx
import { FileTree } from "@canonical/react-ds-app-launchpad";

<FileTree
  data={directoryStructure}
  onSelect={handleFileSelect}
  expandedPaths={["src", "src/components"]}
/>
```

## GitDiffViewer

Display code changes with syntax highlighting:

```tsx
import { GitDiffViewer } from "@canonical/react-ds-app-launchpad";

<GitDiffViewer
  diff={unifiedDiff}
  mode="split"
  language="typescript"
/>
```
