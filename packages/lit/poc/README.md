# @canonical/lit-poc

Universal Web Components built with Lit for the Pragma design system. This package provides foundational UI elements that apply across all Canonical web applications: buttons, badges, cards, tooltips, and more.

## Migration Notice

This package was renamed from `@canonical/webcomponents-ds-global` to `@canonical/lit-poc`.

Update your dependency and imports:

```bash
npm uninstall @canonical/webcomponents-ds-global
npm install @canonical/lit-poc
```

```ts
// Before
import "@canonical/webcomponents-ds-global";

// After
import "@canonical/lit-poc";
```
